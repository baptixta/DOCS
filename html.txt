------ HTML ------
- Estrutura Básica
<!doctype html> => Especificação para o Browser interpretar o documento
<html></html> => Tudo fica dentro dessa tag
<head></head> => "Cabeça" da página
<meta charset="utf-8"> => Encoding da página
<title></title> => Título da página
<link rel="stylesheet" type="text/css" href="css/style.css"> => Chamada do CSS externo

<body></body> => "Corpo" da página
